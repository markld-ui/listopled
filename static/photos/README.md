# static/photos

Эта папка предназначена для локального хранения брендовых материалов проекта LISTOPLED:

- фотографии пледов;
- фотографии мастера;
- логотипы;
- дизайн-референсы;
- будущие материалы из Figma;
- исходники и экспортированные изображения.

По умолчанию содержимое этой папки не коммитится в git, потому что материалы могут быть приватными, тяжёлыми или не предназначенными для публикации.

В репозитории должны оставаться только:

- `.gitkeep`
- `README.md`

Если какой-то конкретный файл нужно использовать в публичном frontend-бандле, его нужно отдельно подготовить, оптимизировать и перенести в согласованную публичную директорию frontend после ревью.
