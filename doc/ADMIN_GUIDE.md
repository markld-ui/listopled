# Admin Guide

Этот файл описывает будущий пользовательский справочник для мастера. В Фазе 1 это только каркас документации: рабочая админ-панель, авторизация и управление данными не реализуются.

## Будущие разделы

- Вход в админку.
- Главная панель.
- Обращения.
- Галерея.
- Калькулятор.
- Отзывы.
- FAQ.
- Контакты.
- Аналитика.
- Настройки.

## Правило Фазы 1

До ревью Фазы 1 не создавать рабочие сценарии админки. Разрешены только структура проекта, placeholders и документация.
