# Domain

## Основные правила

- MVP работает с обращениями, а не с заказами.
- Цена является предварительной и считается только на backend.
- Финальная цена, сроки, доставка и оплата обсуждаются лично с мастером.
- Обращение может содержать параметры калькулятора, но backend должен пересчитать цену перед сохранением.
- Оплата, корзина и личный кабинет клиента запрещены в MVP.

## Public conversion journey

Фаза 2 описывает первый публичный конверсионный срез:

```text
Пользователь открывает сайт → видит красивый лендинг → понимает продукт → смотрит работы → считает примерную цену → получает CTA → может написать мастеру.
```

Доменные правила этого среза:

- галерея, отзывы, FAQ и контакты работают вместе с калькулятором как части одного пути пользователя;
- калькулятор помогает принять решение, но не заменяет личное обсуждение с мастером;
- рассчитанная цена является предварительной;
- CTA после расчета должен вести к личному диалогу, а не к оплате, корзине или клиентскому кабинету;
- будущая форма обращения должна передавать параметры расчета на backend, где цена пересчитывается перед сохранением.

## Калькулятор

Формула:

```text
Итого = F + T * R + B - I
```

Где:

- `R` — коэффициент размера;
- `T` — стоимость ткани;
- `F` — доплата за форму листа;
- `B` — базовая фиксированная часть;
- `I` — скидка.

Начальное значение:

```text
B = 4000
```

Обязательный тест:

```text
Размер: Взрослый 150х220, R = 5
Ткань: Флис, T = 700
Форма: Дуб, F = 300
База: B = 4000
Скидка: I = 300
Итого = 300 + 700 * 5 + 4000 - 300 = 7500 ₽
```

### Доменные сущности калькулятора

Шаг 2.2 реализует в `Listopled.Domain` только доменные сущности и инварианты:

- `BlanketSize`;
- `Fabric`;
- `LeafShape`;
- `Discount`;
- `PriceCalculationSettings`;
- `PriceQuote`;
- `DomainException` для нарушения доменных инвариантов.

Общие правила:

- справочники калькулятора имеют `IsActive` и `SortOrder`;
- денежные значения и коэффициенты используют `decimal`;
- обязательные названия не могут быть пустыми;
- коэффициент размера должен быть больше `0`;
- цены, доплаты, скидки, базовая часть и итоговая цена не могут быть отрицательными;
- `PriceQuote` хранит только параметры и результат предварительного расчета, без персональных данных клиента, статуса заказа или оплаты.

Расчет `7500 ₽` не реализуется на шаге 2.2. Он относится к сервису расчета цены в шаге 2.3.

- Взрослый 150х220 см = 5.
- Подростковый 150х170 см = 4.
- Детский 100х150 см = 2.
- Для новорожденных 80х100 см = 1.6.

- Габардин = 650.
- Флис = 700.
- Трикотаж = 1000.

- Лен = 1200.

Формы листьев:

- Дуб = 300.
- Дуб монгольский = 300.
- Дуб северный = 500.
- Каштан = 500.
- Клен = 500.

- Я новый подписчик Instagram = 300.

## Inquiry status

- `New` — Новая.
- `Seen` — Просмотрена.
- `Contacted` — Связались.
- `InDiscussion` — Обсуждение.
- `Accepted` — Принята.
- `Rejected` — Отклонена.
- `Completed` — Завершена.
